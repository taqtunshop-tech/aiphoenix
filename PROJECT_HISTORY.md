# История проекта AIPhoenix

Этот файл содержит лог изменений проекта для передачи контекста между сессиями.

## [2026-06-24] Разработка концепта презентационного сайта и плана внедрения
**Время:** 16:40
**Описание:**
- Создана структура презентационного сайта (`index.html`, `agents.html`, `rag.html`, `pipeline.html`, `integrations.html`, `hardware.html`).
- Реализован дизайн в стиле карандашного скетча (hand-drawn).
- Сгенерированы и добавлены в проект иллюстрации (теплообменник, сервер, рой агентов, RAG-дерево, пайплайн, интеграции) в едином карандашном стиле.
- Использован предоставленный пользователем логотип (`logo.jpg`).
- Подготовлен подробный план внедрения ИИ-экосистемы на производстве теплообменников "Феникс". Документ включает детальное сравнение Облачной инфраструктуры и Локального сервера (On-Premise), данные для которого были успешно извлечены из базы знаний проекта через **NotebookLM**. План описывает 7 этапов: от закупки серверов за €20K до полноценного развертывания мультиагентной системы.
- Разработан системный файл контекста (`AIPHX_SYSTEM_CONTEXT.md`) для инициализации будущей нейросети. Утверждено короткое стильное название проекта: **AIPHX**. Файл содержит жесткие инструкции (без галлюцинаций, строгий RAG, 100% On-Premise) для всех четырех ролей агентов.

## [2026-06-26] Исправление 3D-вьювера в калькуляторе
**Время:** 03:26
**Описание:**
- Исправлено отображение логотипа на шильдике: внедрен синхронный фолбэк на canvas с текстом "ФЕНИКС" и корректная асинхронная загрузка `logo.png` (с сохранением пропорций, отрисовкой на canvas и назначением как текстуры `MeshBasicMaterial`).
- Удалены артефакты тени по центру теплообменника: отключены свойства `castShadow` и `receiveShadow` у всех мешей.
- Поправлено позиционирование камеры: увеличен радиус (`sph.r=Math.max(3.0, Math.max(C.frameW, C.frameH) * 4.0)`), FOV повышен до 35°, а центровка изменена на `camera.lookAt(0, -C.frameH*0.25, 0)` для полного охвата модели. Те же параметры теперь применяются при смене модели теплообменника.
